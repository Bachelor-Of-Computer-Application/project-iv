# Game Hub User Manual

## Starting the Application

1. Start MySQL Server.
2. Run the database script in `database/game_hub_schema.sql`.
3. Compile and run the Java application using the commands in `README.md`.
4. Log in with the demo account or register a new player.

## Registering

Enter a username, full name, and password. The password must be at least 6 characters and include a number.

## Logging In

Enter your username and password, then choose Login. After a successful login, the dashboard opens.

## Dashboard

The dashboard shows:

- Player profile name.
- Total available games.
- Highest scores in all games.
- Quick launch buttons for each game.
- Sidebar navigation for dashboard, leaderboard, reports, games, and logout.

## Blue Dino Runner

Press Space to jump over obstacles. Your score increases while you survive. When the dino hits an obstacle, the score is saved automatically.

## Snake Game

Use the arrow keys to move. Eat food to grow and increase score. The game ends when the snake hits a wall or itself, then the score is saved.

## Tic Tac Toe

Click a square to place X. The computer places O. Wins, losses, draws, and score are saved after the game ends.

## Leaderboard

Open Leaderboard from the sidebar. Use the filter to view all games or one specific game. The page displays the top 10 records.

## Reports

Reports show total games played, highest score per game, most played game, and recent history.

## Logout

Choose Logout from the sidebar to return to the login screen.
